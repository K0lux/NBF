# NBF Nexus Project Overview

## Description
Integrated web platform for managing trainee lifecycles at New Brain Factory (NBF).

## Modules
- Trainee Management
- Scheduling (Master Schedule)
- Attendance (QR, Geofencing)
- Request Workflows
- AI Companion (RAG)

## Key Rules
- QR codes expire in 2 hours.
- Geofencing tolerance is 200 meters.
- Roles: Admin, Trainee.

## Technologies
- Next.js 16.1
- Supabase
- Clerk
- Vercel AI SDK
